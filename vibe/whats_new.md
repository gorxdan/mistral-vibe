# What's new in v0.1.0
- **LSP code intelligence**: semantic go-to-definition, references, hover, and call hierarchy that resolve imports, overloads, and generated code
- **Typed memory**: an always-on memory index with auto-extraction and per-project scoping, so saved facts stay scoped to their project
- **Le Chaton mode**: maximum thinking plus automatic multi-agent workflow orchestration for large or cross-checked tasks
- **Sakana Fugu models**: bundled Fugu and Fugu-ultra provider presets, including a 1M-context compaction budget
- **Git-tag versioning**: releases now derive their version from `vX.Y.Z` git tags via hatch-vcs
